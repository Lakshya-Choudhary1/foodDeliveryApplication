# 🍽️ Zomato Clone – MERN Stack with Reels-Style Video Feed

A **full-stack Zomato-inspired food discovery platform** built with the **MERN stack**.
The project includes a **modern restaurant browsing experience**, **reels-style food video feed**, and **responsive UI** for discovering restaurants and food content.

## ✨ Features

* 🍔 **Restaurant Discovery** – Browse restaurants with detailed information
* 🎥 **Reels-Style Video Feed** – Short food videos similar to social media reels
* 🔍 **Search & Filter** – Find restaurants based on cuisine or preferences
* ⭐ **Ratings & Reviews System**
* 🖼️ **Restaurant Image Uploads** using Cloudinary
* 🧾 **RESTful APIs** built with Node.js and Express.js
* 🧱 **Database Management** using MongoDB
* 🎨 **Modern Responsive UI** with React and Tailwind CSS
* 🧠 **Efficient State Management** for dynamic UI updates
* 📱 **Mobile-Friendly Design** optimized for reels-style scrolling

## 🛠️ Tech Stack

**Frontend**

* React
* Tailwind CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

**Other Tools**

* Cloudinary
* Git & GitHub
